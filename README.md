# 🥤 Fly to Cart Animation

An elegant, modern, and high-performance **Fly to Cart** micro-interaction built using vanilla HTML, CSS, and JavaScript. This project replicates a premium e-commerce product card experience where the product image dynamically clones, shrinks, and flies into the header cart upon clicking the "Add to Cart" button.

---

## ✨ Features

* **Dynamic Trajectory:** Uses JavaScript `getBoundingClientRect()` to calculate accurate screen coordinates, ensuring the flying effect lands perfectly on the cart icon regardless of screen size.
* **Parabolic Feel:** Smooth curve transition using custom CSS `cubic-bezier` timing functions.
* **Modern UI/UX:** Clean card layout with drop shadows, crisp typography, and responsive scaling.
* **Micro-interactions:** Interactive "bump" animation on the cart badge and smooth button active states.
